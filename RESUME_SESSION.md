## 晨风清兴 / AskString AI — 挖掘记录

这是时间最早的一段经历（2023.04 - 2024.02），也是原来简历里写得最单薄的一块，现在正在挖。

**背景（已确认）**

ChatGPT 爆火的大背景下，Text-to-SQL 技术兴起，但早期的问题在于用户需要精准的结构化表述才能有效查询，而实际上不懂 SQL 的人不会用自然语言问得精准，懂 SQL 的人又不需要这么查。AskString 想解决的是这个矛盾，让营销、运营等非技术用户也能做以往只有专业数据分析师才能完成的聚合查询和根因分析。

产品背后有学术支撑：华盛顿大学 Meliou & Suciu 团队 2010 年的因果性数据库系列研究（"Why so? or Why no?"），用 Halpern-Pearl 因果框架解释 SQL 查询的根因，这是 AskString 因果分析能力的理论来源。

**团队（已确认）**

4 人精英团队：CEO（CMU 机器学习背景）、COO、PM、Francis（唯一工程师）。Francis 和 CEO 直接打配合——CEO 提出算法构想，Francis 负责工程落地。

**技术方案（已确认）**

核心是「LLM Function Sandbox」模式，而不是传统的直接生成 SQL 字符串：
- 用户提问 → LLM 拆解成多个子任务
- 每个子任务在 Serverless Python 沙箱里执行（部署在 Railway），生命周期为会话级
- 执行结果回流进 LLM 上下文，驱动下一步推理（ReAct：执行 → 观察 → 反思）
- 先「发散」遍历数据关联，再「收敛」聚合到因果根因

这在 GPT-o1 和成熟 CoT 出现之前，本质上是手工通过提示词工程模拟链式推理，DFS 式数据探索是对这个过程的一个准确比喻。

**成果（已确认）**

- 产品 2023 年 6 月中旬上线
- 10+ 家海外企业客户，主要是管理层用户
- 年费订阅席位制，$200/席位/年
- ARR $100k+ 美元
- 拿到国内知名投资机构百万美元天使轮

**基础设施（已确认）**

容器编排是最有挑战的一块。难点在于完全没有运维和 CI/CD 背景，但需要独立搭起云端自动化部署体系。解法是借助早期 AI 联网检索能力快速上手，基于 GitHub Actions + Docker Compose 实现了全自动化 CI/CD 流程，同时用统一的 CLI 脚本和配置文件管理多个 SaaS 平台——而不是每个平台分别手动操作 dashboard。结果是节省了一个专职运维岗的开支，并实现了多平台的统一聚合管理。

SSO 企业级单点登录 + Stripe 支付系统对接也是基础设施的一部分（细节待补充）。

**SSO & Stripe（已确认）**

SSO 用 Clerk + OAuth，核心价值是复用企业客户已有的 AWS/Google 身份体系，打通数据仓库接入和运营数据 drain，是选型决策而非技术难点。

Stripe 标准集成，$200/席位/年的年费订阅模型。难点在于 Webhook 的三环境测试（dev → preview → prod 严格区分）。

**质量保障体系（已确认，信息量最大的部分）**

搭了一套超出团队规模的工程质量体系：
- Playwright 全自动 E2E 测试，覆盖完整用户旅程（Landing Page → 注册登录 → 使用 → 关闭），多套用户故事测试管道，边缘情况全覆盖
- 测试指标 + 埋点数据接入自建 LLM 评估管道，自动分析性能/UX 问题，自动上报飞书
- Sentry 生产监控，24/7 Bug 自动检测，热修复机制，保障 99%+ 可用性
- GitHub Issue 自动归因：LLM 追溯 Bug 至具体提交，自动 @提交者

## Metasight 元见 — 挖掘记录

**产品定位（已确认）**

AI 命理产品，面向深夜焦虑的 Z 世代。传统命理软件审美太差（红底黄字、满屏大吉凶煞），而这一代用户同时有审美要求和情感陪伴需求。产品理念：破迷雾，见真我——不是预测命运，而是陪伴用户看清自己。

**技术架构（已确认）**

三层设计，核心是明确区分 LLM 的能力边界：
1. **历法引擎层**：八字排盘、大运流年、十神、纳音等确定性数据，通过历法直接计算，不依赖 LLM
2. **量化层**：以加权平均算法将传统命理口诀和模糊规则数字化为结构化数据
3. **AI 解读层**：将结构化命盘数据作为上下文注入 LLM，模型只负责自然语言解读，规避幻觉

**部署与增长（已确认）**

- Vercel serverless 部署，全球边缘加速
- SEO：针对繁体中文市场做多语言优化，Google「AI 命理」关键词排名第 6，Bing 日均曝光 200+
- 社区：2026 年 1 月中旬在观猹社区连续 1 周登顶新品榜第一，获多名金牌 Hunter 好评，口碑裂变传播

**数据（已确认）**

- 注册 2000+，付费用户 150+，MRR ¥5000+
- 两周内 DAU 破 200
- 付费用户反馈：AI 对话体验与命理准确度业内最佳

**已确认的决策**

- AI 对话层无特别工程化工作，不写
- 用户口碑反馈（业内最佳）不写入简历（太主观）
- Vercel edge 一句话带过，不单独列为技术维度
- SEO 归入成果，不作为独立技术维度
