# Portfolio

> 个人简历展示网页

## 技术栈

- Astro + React
- Tailwind CSS v4
- Motion (动画)
- Base UI (组件原语)
- TypeScript

## 开发

```bash
bun install
bun run dev
```

## 构建

```bash
bun run build
```

## 目录结构

```
src/
├── components/     # 组件
├── content/        # 简历数据
├── i18n/           # 国际化
├── layouts/        # 布局
├── pages/          # 页面
└── styles/         # 样式
```
